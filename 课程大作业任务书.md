# 《智能语音处理》课程大作业任务书

## 题目1：自动语音识别系统

- **目标**：构建一个自动语音识别系统，输入为音频，输出为文本
- **数据集**：THCHS30
- **测试技术指标**：CER（字错误率，越小越好）
- **提交**：
  1. 项目报告（需完整清晰描述技术方案、算法设计、实验结果与分析）
  2. 完整可复现代码（含数据准备脚本、环境依赖、训练/测试配置、无需数据集）
  3. 预训练模型
  4. 3条测试音频与识别结果

## 题目2：语音合成系统

- **目标**：构建一个文本到语音合成系统，输入为文本，输出为语音
- **数据集**：LJ Speech Dataset
- **测试技术指标**：PESQ（感知语音质量，越大越好）
- **提交**：
  1. 项目报告（需完整清晰描述技术方案、算法设计、实验结果与分析）
  2. 完整可复现代码（含数据准备脚本、环境依赖、训练/测试配置、无需数据集）
  3. 预训练模型
  4. 3条测试文本与合成音频
